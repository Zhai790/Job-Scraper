📁 Location: D:\job-board-monitor

🎯 What is this?
   Automated system that checks Lever job boards daily and emails
   you when junior/entry-level positions are posted.

📋 Next Steps:

   1️⃣  CONFIGURE COMPANIES (Required)
      Edit monitor.py → Change COMPANIES list
      Example: ["magnetforensics", "shopify", "1password"]

   2️⃣  PUSH TO GITHUB (Required)
      • Create new repo at github.com/new
      • Run: git remote add origin https://github.com/YOU/job-board-monitor.git
      • Run: git push -u origin main

   3️⃣  ADD EMAIL SECRETS (Required)
      GitHub repo → Settings → Secrets → Actions
      Add 5 secrets:
         - SMTP_SERVER (e.g., smtp.gmail.com)
         - SMTP_PORT (e.g., 587)
         - SMTP_USER (your email)
         - SMTP_PASSWORD (Gmail app password)
         - RECIPIENT_EMAIL (where to send alerts)

   4️⃣  ENABLE GITHUB ACTIONS (Required)
      • Actions tab → Enable workflows
      • Settings → Actions → General
        → Enable "Read and write permissions"

   5️⃣  TEST IT! (Recommended)
      • Actions tab → Job Board Monitor
      • Click "Run workflow"
      • Check logs and email inbox

📚 Full Documentation:
   • SETUP.md - Detailed setup instructions

   • config.example.env - Local testing template

🔧 Customization:
   • Keywords: Edit KEYWORDS in monitor.py
   • Schedule: Edit cron in .github/workflows/job-monitor.yml
   • Location: Edit LOCATION in monitor.py
